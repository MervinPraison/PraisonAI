# Templates CLI

```bash
praisonai templates list
```

```bash
praisonai templates search video
```

```bash
praisonai templates info ai-video-editor
```

```bash
praisonai templates run ai-video-editor --input video.mp4 --output edited.mp4
```

```bash
praisonai templates run ai-video-editor --input video.mp4 --output edited.mp4 --offline
```

```bash
praisonai templates run ai-video-editor --input video.mp4 --output edited.mp4 --strict-tools
```

```bash
praisonai templates run ai-video-editor --input video.mp4 --tools ./my_tools.py
```

```bash
praisonai templates run ai-video-editor --input video.mp4 --tools-dir ./tools
```

```bash
praisonai templates run ai-video-editor --input video.mp4 --tools-source praisonai_tools.video
```

```bash
praisonai templates run ai-video-editor --input video.mp4 --no-template-tools-py
```

```bash
praisonai templates init my-project --template ai-video-editor
```

```bash
praisonai templates cache clear
```

```bash
praisonai templates cache list
```
