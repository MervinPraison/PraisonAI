# C9.1 Package Boundaries — praisonai-bot

Four-tier Python stack:

```
praisonaiagents → praisonai-code + praisonai-bot → praisonai (wrapper)
```

## Dependency rules

| Package | May import | Must not (PyPI dep) |
|---------|------------|---------------------|
| `praisonaiagents` | stdlib, core deps | praisonai, praisonai-code, praisonai-bot |
| `praisonai-code` | praisonaiagents | praisonai, praisonai-bot (use `_bot_bridge`) |
| `praisonai-bot` | praisonaiagents | praisonai, praisonai-code (use `_code_bridge` / `_wrapper_bridge`) |
| `praisonai` | all three | — |

## Two daemons

| Command | Package |
|---------|---------|
| `praisonai daemon` | praisonai-code (warm runtime) |
| `praisonai-bot` OS service | praisonai-bot (`daemon/`) |

## Shim paths (frozen)

- `praisonai.bots` → `praisonai_bot.bots` (`alias_package`)
- `praisonai.gateway` → `praisonai_bot.gateway`
- `praisonai.cli.commands.{bot,gateway,...}` → `praisonai_bot.cli.commands.*`
