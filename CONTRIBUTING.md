# Contributing to PraisonAI

Thank you for your interest! Here's how to get started:

## Getting Started

1. Fork the repository
2. Clone your fork and install dependencies:
   ```bash
   git clone https://github.com/your-username/PraisonAI.git
   cd PraisonAI
   pip install -e src/praisonai-agents/
   pip install -e src/praisonai/
   ```
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Submit a pull request

## Guidelines

- Follow existing code style (PEP 8)
- Add tests for new features and run them locally using pytest:
  ```bash
  cd src/praisonai-agents && pytest tests/
  ```
- Update documentation as needed
- Check existing issues or open a new one to discuss your idea before implementing large changes

For questions or support, please visit our [GitHub Issues](https://github.com/MervinPraison/PraisonAI/issues).
